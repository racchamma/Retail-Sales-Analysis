 🛒 Retail Sales Analysis — Python & Power BI

 📌 Project Overview
This project analyzes retail liquor sales data (Wine & Beer) across multiple suppliers.
It includes data cleaning, exploratory analysis in Python, and an interactive Power BI dashboard.

 📁 Files in This Repository

| File | Description |
|------|-------------|
| `retail_analysis.html` | Jupyter Notebook (exported as HTML) — Python EDA & analysis |
| `retail_dashboard.pbix` | Power BI Dashboard — Visual reports & insights |
| `retail_cleaned.csv` | Cleaned dataset (~29,918 rows) — Retail sales data |

 📊 Dataset Details

- **Source**: Retail liquor sales records
- **Rows**: ~29,918
- **Columns**: YEAR, MONTH, SUPPLIER, ITEM CODE, ITEM DESCRIPTION, ITEM TYPE, RETAIL SALES, RETAIL TRANSFERS, WAREHOUSE SALES, MONTH_NAME

🛠️ Tools Used
- Power BI Desktop
- Jupyter Notebook
